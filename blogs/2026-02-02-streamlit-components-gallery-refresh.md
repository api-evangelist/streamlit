<h3>Introduction</h3><p>Streamlit components are what make our ecosystem so powerful — they let you go beyond the built-in widgets and add everything from custom visualizations to authentication flows. But with over 200 community-built components available, finding the right one shouldn’t feel like searching for a needle in a haystack.</p><p>We just shipped a refresh of the Streamlit Component Gallery: <a href="https://streamlit.io/components">https://streamlit.io/components</a></p><figure><img alt="" src="https://cdn-images-1.medium.com/max/1024/1*B-mAAg-t_rcvOkc208YzPA.png" /></figure><h3><strong>What’s new</strong></h3><ul><li>A cleaner layout</li><li><strong>Search + sorting</strong> to make it easier to discover components and get to the one you need faster</li></ul><h3><strong>Why we did this</strong></h3><p>The gallery is one of the first places people go to extend Streamlit, so we wanted it to be easier to browse and easier for the community to keep growing.</p><h3><strong>Behind the scenes</strong></h3><p>We also reworked how the gallery is powered. It now runs on a <strong>new open source registry</strong>, which makes community submissions much simpler. If you’ve built a Streamlit component, you can now get it listed with a pull request.</p><h3><strong>Have a component to share?</strong></h3><p>We’d love to include it.<br />Add your component via PR to the registry by following the submission instructions here: <a href="https://github.com/streamlit/gallery/tree/main/components/registry#readme">https://github.com/streamlit/gallery/tree/main/components/registry#readme</a></p><p>Can’t wait to see what you all build!</p><img alt="" height="1" src="https://medium.com/_/stat?event=post.clientViewed&amp;referrerSource=full_rss&amp;postId=e27a890e7229" width="1" /><hr /><p><a href="https://blog.streamlit.io/streamlit-components-gallery-refresh-e27a890e7229">Streamlit Components Gallery Refresh</a> was originally published in <a href="https://blog.streamlit.io">Streamlit</a> on Medium, where people are continuing the conversation by highlighting and responding to this story.</p>
