<figure><img alt="" src="https://cdn-images-1.medium.com/max/1024/1*BDucTha9h-_hM9-LcXBZxA.png" /></figure><h4>Experiment with this open-source LLM from Meta</h4><p>Generative AI has been widely adopted, and the development of new, larger, and improved LLMs is advancing rapidly, making it an exciting time for developers.</p><p>You may have heard of the recent release of <a href="https://ai.meta.com/llama/?ref=streamlit.ghost.io">Llama 2</a>, an open-source large language model (LLM) by Meta. This means that you can build on, modify, deploy, and use a local copy of the model, or host it on cloud servers (e.g., <a href="https://replicate.com/?ref=streamlit.ghost.io">Replicate</a>).</p><p>While it’s free to download and use, it’s worth noting that self-hosting the Llama 2 model requires a powerful computer with high-end GPUs to perform computations in a timely manner. An alternative is to host the models on a cloud platform like Replicate and use the LLM via API calls. In particular, the three Llama 2 models ( llama-7b-v2-chat, llama-13b-v2-chat, and llama-70b-v2-chat) are hosted on Replicate.</p><p>In this post, we’ll build a Llama 2 chatbot in Python using Streamlit for the frontend, while the LLM backend is handled through API calls to the Llama 2 model hosted on Replicate. You’ll learn how to:</p><ol><li>Get a Replicate API token</li><li>Set up the coding environment</li><li>Build the app</li><li>Set the API token</li><li>Deploy the app</li></ol><p>🦙 Want to jump right in? Here’s the <a href="https://llama2.streamlit.app/">demo app</a> and the <a href="https://github.com/dataprofessor/llama2">GitHub repo</a>.</p><h3>What is Llama 2?</h3><p>Meta released the second version of their open-source Llama language model on July 18, 2023. They’re democratizing access to this model by making it free to the community for both research and commercial use. They also prioritize the transparent and responsible use of AI, as evidenced by their <a href="https://ai.meta.com/llama/responsible-use-guide?ref=streamlit.ghost.io">Responsible Use Guide</a>.</p><p>Here are the five key features of Llama 2:</p><ol><li>Llama 2 outperforms other open-source LLMs in benchmarks for reasoning, coding proficiency, and knowledge tests.</li><li>The model was trained on almost twice the data of version 1, totaling 2 trillion tokens. Additionally, the training included over 1 million new human annotations and fine-tuning for chat completions.</li><li>The model comes in three sizes, each trained with 7, 13, and 70 billion parameters.</li><li>Llama 2 supports longer context lengths, up to 4096 tokens.</li><li>Version 2 has a more permissive license than version 1, allowing for commercial use.</li></ol><h3>App overview</h3><p>Here is a high-level overview of the Llama2 chatbot app:</p><ol><li>The user provides two inputs: (1) a Replicate API token (if requested) and (2) a prompt input (i.e. ask a question).</li><li>An API call is made to the Replicate server, where the prompt input is submitted and the resulting LLM-generated response is obtained and displayed in the app.</li></ol><figure><img alt="" src="https://cdn-images-1.medium.com/max/1024/0*9b2FvOJb3LUIlY3U.jpg" /></figure><p>Let’s take a look at the app in action:</p><ol><li>Go to <a href="https://llama2.streamlit.app/">https://llama2.streamlit.app/</a></li><li>Enter your Replicate API token if prompted by the app.</li><li>Enter your message prompt in the chat box, as shown in the screencast below.</li></ol><figure><img alt="" src="https://cdn-images-1.medium.com/max/1024/0*qrKWgQqBQF3pi4Zl.gif" /></figure><h3>1. Get a Replicate API token</h3><p>Getting your Replicate API token is a simple 3-step process:</p><ol><li>Go to <a href="https://replicate.com/signin/?ref=streamlit.ghost.io">https://replicate.com/signin/</a>.</li><li>Sign in with your GitHub account.</li><li>Proceed to the API tokens page and copy your API token.</li></ol><figure><img alt="" src="https://cdn-images-1.medium.com/max/1024/0*VvPdfKQgLQDLwIUX.png" /></figure><h3>2.
